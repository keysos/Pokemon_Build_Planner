# Pokémon Team Planner

<p align="center">
  <img src="./assets/project_logo.png" alt="Pokemon Team Builder" width="800">
</p>
A web-based Pokémon team builder that allows users to create and customize a Pokémon party with abilities, held items, and moves.

## Features

- Build a team of up to 6 Pokémon
- Select Pokémon from Generation 1 to 5
- View Pokémon sprites and details
- Choose abilities, held items, and moves
- Prevent duplicate moves within the same Pokémon
- Responsive layout for desktop, tablet, and mobile
- Animated Pokémon sprite previews

## Technologies Used

- HTML5
- CSS3
- JavaScript
- PokéAPI

## API

This project uses the PokéAPI to fetch:
- Pokémon data
- Abilities
- Moves
- Sprites

https://pokeapi.co/
